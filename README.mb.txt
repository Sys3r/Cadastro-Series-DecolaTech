# Primeiro projeto do bootcamp DecolaTech 2021



##### Criando um APP simples de cadastro de séries em .NET

​	

Esse foi o meu primeiro projeto do programa DecolaTech 2021. Apesar de ainda não dominar o conteúdo, 

consegui acompanhar as aulas e replicar toda a estrutura da programação orientada objeto. No início meu

código teve 142 problemas, mas ao rever diversas vezes as aulas, consegui colocá-lo para debugá-lo com sucesso.

A partir desse projeto, irei estudar mais sobre as estruturas e me aprofundar no assunto a fim de modificar algumas linhas

e tornar o código mais com a minha "cara".
